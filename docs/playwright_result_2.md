## テスト7: 組織管理

**時刻**: テスト6直後

### 手順
1. `/relationships` にアクセス
2. 「追加」ボタンをクリック
3. ノード名「株式会社サンプル」を入力
4. 「追加」ボタンをクリック

### 結果
❌ **FAIL** - ランタイムエラー発生

### エラー詳細
```
Runtime TypeError: Cannot read properties of undefined (reading 'length')
  at src/components/relationship/RelationshipTree.tsx (37:37)
  > const hasChildren = node.children.length > 0;
```

### 原因
`node.children` が `undefined` の状態で `.length` にアクセスしようとしている。APIから返されるノードデータに `children` プロパティが含まれていない可能性。

### 修正が必要なファイル
- `frontend/src/components/relationship/RelationshipTree.tsx` - nullチェック追加
- または `backend/app/api/v1/endpoints/relationships.py` - レスポンスにchildrenを含める
